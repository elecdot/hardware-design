# JY901

Vendor documentation, tools, PCB/library files, and example programs for the JY901 IMU module are stored here as reference material.

## Index

| Content | Purpose |
|---|---|
| Vendor PC tool directories | PC-side tools and runtime files. |
| Vendor example-code directories | Example code for 51, Arduino, STM32, and Windows targets. |
| Vendor PCB/library directories | PCB and library assets. |

Do not treat vendor examples as repo coding style. Extract protocol facts into [../i2c_axi_mpu9250.md](../i2c_axi_mpu9250.md), [../wiring.md](../wiring.md), or module README files before using them in project code.
