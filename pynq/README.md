PYNQ board-side demo code, drivers, and notebooks live here.

## Index

|---|---|

The current target board runtime is Python 2.7.10 on Linux 4.6.0-xilinx, so
board-side Python should stay compatible with Python 2.7 unless this runtime is
upgraded.
